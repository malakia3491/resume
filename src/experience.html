Дипломная работа: Подсистема контроля выкладки товаров на полках магазина

Репозиторий проекта | Dataset на Kaggle

Ситуация: В рамках дипломной работы требовалось выполнить техническое задание, спроектировать и разработать подсистему видеоаналитики для детекции и классификации товаров на полках магазинов. Проект был направлен на снижение количества отсутствующих товаров и ускорение реакции персонала.

Задача: Создать полноценный пайплайн обработки изображений, от получения данных до инференса ML-модели, с разработкой backend, frontend и интеграцией с внешними сервисами.

Действие:

  • Реализовал пайплайн обработки изображений: от получения данных до инференса модели (YOLO).
  • Разработал backend на Python с использованием FastAPI, SQLAlchemy и PostgreSQL; использовал ultralytics (YOLO), PyTorch, scikit-learn и OpenCV.
  • Разработал пользовательский интерфейс на Vue.js и TypeScript.
  • Выполнил интеграцию с Telegram API для отправки уведомлений с помощью aiogram.
  • Реализовал приложение на PyQt5 для отправки изображений на backend.

Результат: Успешно спроектирована и разработана рабочая подсистема видеоаналитики, демонстрирующая высокую эффективность в детекции и классификации товаров, способная улучшить операционные процессы магазинов.
